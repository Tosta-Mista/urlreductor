urlreductor
===========

Simple url reductor written in django.

## Install
Clone the repo :
```
git clone git@github.com:Tosta-Mixta/urlreductor.git
```

Sync Database :
```
cd urlreductor
python manage.py syncdb
```

Run dev server :
```
python manage.py runserver
```

Connect to the webserver :<br />
http://127.0.0.1:8000/
