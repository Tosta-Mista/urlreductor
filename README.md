urlreductor
===========

Simple url reductor writed in django.

## Install
Clone the repo :
```
git clone git@github.com:Tosta-Mixta/urlreductor.git
```

Sync Database :
```
cd urlreductor
```
```
python manage.py syncdb
```

Run dev server :
```
python manage.py runserver
```
